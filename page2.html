Startside Prosjekter

INTRODUKSJON HTML (U2-OBLIGATORISK INNLEVERING)

  • Sett opp et prosjekt i StackBlitz
  • Koble prosjektet til Github og opprett et GitHub repository
  • Bruk semantisk HTML som header, main, footer.
  • Bruk elementer som h1-2, p, ul/li og img til å vise innhold.
  • Publisere repositoriet via Netlify/Vercel for å få en nettside
  • Legg til CSS for å legge til litt styling til nettsiden, som for eksempel endre farge og størrelse.

INTRODUKSJON TIL FIGMA (U6-OBLIGATORISK INNLEVERING)

  • Lage wireframe til en nettside i desktop (Velg frames -> desktop -> desktop/wireframe)
  • Sett opp en header som har en navigasjonsbar
  • Sett opp main som har ulike seksjoner med innhold
  • Sett opp en footer som har kontakt informasjon

MEDIA GALLERI (U11-OBLIGATORISK INNLEVERING)

  • Bruk semantisk HTML
  • Bruk plasserings metoden grid og/eller flex for å sette opp innholdet.
  • Lag et design i CSS som står i samsvar med innholdet i HTML med farger og fontfamilie
  • Sett opp at nettsiden har et bryte punkt for stor og liten skjerm

HTML FORM & UNIT CONVERTER (U16-OBLIGATORISK INNLEVERING)

  • Opprett en HTML-struktur som følger beste praksis for semantisk strukturering av innholdet
  • Benytt Form-elementene i HTML for å utforme et brukervennlig skjema
  • Implementer JavaScript-funksjoner som reagerer på brukerens handlinger
  • Utfør konverterings beregningene basert på de valgte enhetene i brukerens input
  • Presenter resultatet av konverteringen tydelig på siden
  • Sett opp styling til applikasjonen
